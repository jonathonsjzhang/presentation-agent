# 信息充分性

## 什么算「信息量充分」

一页不能只有结论和一个数字。至少要考虑三个层次：

### 层次 1：主证据（must have，直接支撑 page_claim）
- 核心量化数据（如留存率、占比、变化幅度）
- 带具体数值，不能是「显著提升」「明显更高」类概括

### 层次 2：辅助证据（should have，帮助理解主证据）
- 对比基线（同行 / 历史同期 / 对照组）
- 拆解维度（按人群 / 功能 / 渠道 / 时间）
- 机制解释（为什么是这样，用户原声可在此）
- 趋势（变化方向、拐点、可持续性判断）

### 层次 3：边界信息（must have，决定结论能用在哪）
- 定义与口径（指标如何计算）
- 样本量与时间窗
- 统计显著性 / 置信区间
- 局限性 / 不能排除的替代解释
- 对结论适用范围的限定

## 信息不足时的原则

| 情况 | 做法 |
|---|---|
| 只有主证据，缺辅助/边界 | 结论强度降一档，标记缺口 |
| 连主证据都不具体 | 结论强度必须降为 hypothesis |
| 缺对比基线 | 不能做「领先/优于」类判断 |
| 缺机制解释 | 不能做因果推断 |
| 缺边界信息 | 必须在 caveat 中标明「据此不能回答什么问题」 |

## 反例（AI 版常见）

```
「纯白用户留存表现突出，显著高于非纯白用户」

问题：
- 没有具体数值，无法判断「突出」的程度
- 没有对比基线，无法判断是绝对值还是相对价值
- 没有样本量，无法判断可信度
- 没有机制解释，无法判断是否可以复制到其他场景
```

## 正例（人工作版范例）

```
「元宝纯白用户 30 日强留存 27%（n=1,243，±3pp）
  同期非纯白用户 12%（n=3,871，±2pp）
  差值 +14pp，统计显著 p < 0.01
  拆解：纯白用户留存高主要来自学校/单位线下场景（45%）带来的固定使用场景绑定
  caveat：样本仅覆盖 Q2，开学季场景效应可能放大差异」
```
