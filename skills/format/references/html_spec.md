# §H-1 format.html 专用规则（占位）

> **状态**：v3 当前阶段 html 载体**未实测过**，本文件仅做占位说明。**禁止**在 html 能力上声称通过 v0.9.1 验证。

## 1. 能力定位

交互网页（可展开、可分享、外发必脱敏；响应式 + 暗色主题）。

## 2. 与 format.ppt 的差异

| 维度        | PPT                     | HTML              |
| --------- | ----------------------- | ----------------- |
| 字号单位      | pt                      | px                |
| 字号 title  | 18pt                    | 32px              |
| 字号 body   | 9.2pt                   | 16px              |
| 字号 footnote | 7.8pt                  | 12px              |
| 行距        | 1.0                     | 1.6               |
| margin    | 0.55 inch               | 24px              |
| 栏数       | 12 网格                   | 12 网格             |
| 表格       | 无斑马纹                   | sticky top + 斑马纹  |
| Logo      | 角落                      | 顶部 nav 旁         |
| 动画       | 克制                      | 完整（折叠/展开 ≤ 0.3s）|
| 响应式      | 无                       | 6 断点             |
| 主题切换     | 无                       | light / dark 双主题  |
| 脱敏       | 仅内部                     | **外发强制**          |
| 键盘导航     | 无                       | **强制**            |
| 加载性能     | 离线                      | LCP ≤ 2.5s        |

## 3. Layout 库（待实测）

v3 旧版罗列的 html layout 在 v3 阶段未实测。**禁止**声称这些 layout 通过验证。
