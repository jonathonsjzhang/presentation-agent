# HTML carrier

Use only when `output_format=html`.

- The core Storyline and Report remain section-based; the renderer, not an upstream worker, translates them into browser modules and navigation.
- Preserve the approved argument order, wording, evidence, caveats, Q&A, and appendix hierarchy.
- Format only selects evidence-led visuals; it does not prescribe components, CSS, breakpoints, or interaction behavior.
- Responsive layout, navigation, accessibility, overflow handling, and HTML generation belong to the renderer and runtime quality checks.
